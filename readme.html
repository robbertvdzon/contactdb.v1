Install nodejs:
mkdir ~/src && cd $_
wget -N http://nodejs.org/dist/node-latest.tar.gz
tar xzvf node-latest.tar.gz && cd node-v*
./configure
checkinstall   #remove the "v" in front of the version number in the dialog!
dpkg -i node_*

Build frontend:
su robbert
cd /workspace/msw-frontend
